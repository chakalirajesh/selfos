# SelfOS

SelfOS is an AI-powered Personal Operating System designed to help individuals manage their personal and professional growth from a single platform.

## Features

* Task Management
* Goal Tracking
* Habit Tracking
* Learning Progress Tracking
* Project Management
* Notes Management
* Career Growth Tracking
* Analytics Dashboard

## Tech Stack

### Backend

* Java 21
* Spring Boot 3
* Spring Security
* Spring Data JPA

### Database

* PostgreSQL 16

## Getting Started

### Clone Repository

```bash
git clone https://github.com/Rajesh-C01/selfos.git
cd selfos
```

### Run with Docker

```bash
docker-compose up -d
```

### Deploy to Kubernetes

```bash
kubectl apply -f kubernetes/
```

### Deploy with Helm

```bash
helm install selfos ./selfos-chart -n selfos
```
