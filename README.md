# SelfOS

SelfOS is a personal operating system designed to manage:

- Tasks
- Goals
- Projects
- Learning
- Habits
- Notes
- Career Growth

## Tech Stack

Backend:
- Java 21
- Spring Boot

Database:
- PostgreSQL

DevOps:
- Docker
- Kubernetes
- Helm
- Jenkins
- Prometheus
- Grafana
